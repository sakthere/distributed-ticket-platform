# ADR-005: FluentValidation

Status: Accepted

## Context

Input validation should remain separate from business logic.

## Decision

Use FluentValidation.

Validators execute before handlers.

## Alternatives

DataAnnotations

Pros

Built into ASP.NET

Cons

Limited

Mixed with DTOs

Manual Validation

Rejected

Too repetitive.

## Consequences

Handlers focus only on business rules.
